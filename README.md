# BrewNest

BrewNest is a premium coffee brand website built with Flask.  
The project focuses on clean structure, modern UI, responsive design, and production-ready development practices.

## Features

- Flask application factory pattern
- Responsive modern UI
- Separate stylesheet for each page
- Active navigation state
- Contact form handling
- Flash success and error messages
- Clean project structure
- GitHub-ready workflow

## Pages

- Home
- Menu
- About
- Contact

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
